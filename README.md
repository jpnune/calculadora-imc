# Calculadora de IMC - React & Vite

Este é um projeto desenvolvido para o exercício da EBAC, uma calculadora de IMC (Índice de Massa Corporal) moderna e responsiva.

## 🚀 Tecnologias Utilizadas

- **ReactJS**
- **Vite**
- **CSS3 (Vanilla)**
- **JavaScript (ES6+)**

## 📂 Estrutura do Projeto

- `dev/`: Contém os arquivos de desenvolvimento (`src` e `index.html`).
- `dist/`: Contém os arquivos otimizados para produção (gerados automaticamente).

## 🛠️ Como Executar

### Desenvolvimento
```bash
npm install
npm run dev
```

### Produção (Build)
```bash
npm run build
```

## 🌐 Deploy na Vercel

O projeto está pronto para deploy na Vercel. Ao conectar seu repositório do GitHub:
- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Root Directory**: `./`

## ✨ Funcionalidades

- **Máscara Decimal**: Digite apenas os números e o aplicativo formata como decimal (ex: 175 -> 1,75).
- **Cálculo Automático**: O IMC e a classificação são atualizados em tempo real.
- **Tabela Dinâmica**: A linha correspondente ao seu IMC é destacada na tabela de referência.
- **Design Premium**: Interface escura com gradientes e animações.
